Overview

Purpose

When was the last time you visited a website and left because it was too cluttered and difficult to navigate? I want to create a website that provides necessary information about a non-profit organization that focuses on providing people of all ages recreational activities all year long. This website will be designed so that anyone, even children, can navigate it while still being visually appealing.

Content

Home page

Images: There will be a few images with one main image centered near the top of the site. Every image will be directly related to the organization and some of the activities held. Design: The website will be designed with the organization’s theme in mind. The theme is based on the name Hyperspace. So colors that are often associated with space and neon lights would be used. Links: There will be two links to the child pages. One of them is an “About Us” page that will describe the owners of the organization, their goal, and their motivation. It will also include an image of the owners. The other page is a registration page where the user will be able to sign up and book a time to go to the park. It will contain very little besides the required. Another option instead of a registration page is a FAQ page which would be much easier to code.
